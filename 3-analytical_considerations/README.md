# Analytical Consideration for Longitudinal DNAm

This section contains a script to estimate cell type composition from DNAm data. It includes several algorithms and reference datasets suitable for different age groups and tissue types (whole blood and cord blood).

---

## Section structure

| File | Description |
|------|-------------|
| `3.1-cell_type_proportions.R` | Cell type estimation using [meffil](https://github.com/perishky/meffil/tree/master) and [EpiDISH] (https://github.com/sjczheng/EpiDISH) |
